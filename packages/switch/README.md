# GITS Switch

Vue Switch Component.

## Installation

npm

```
npm i @gits-id/switch
```

yarn

```
yarn add @gits-id/switch
```

pnpm

```
pnpm i @gits-id/switch
```

## Usage

```vue
<script setup lang="ts">
import VSwitch from '@gits-id/switch';
import '@gits-id/switch/dist/style.css';
</script>

<template>
  <VSwitch />
</template>
```
