# Morpheme Popover

Morpheme Popover Component.

## Installation

npm

```
npm i @morpheme/popover
```

yarn

```
yarn add @morpheme/popover
```

pnpm

```
pnpm i @morpheme/popover
```

## Usage

```vue
<script setup lang="ts">
import Dropdown from '@morpheme/popover';
</script>

<template>
  <Dropdown />
</template>
```
