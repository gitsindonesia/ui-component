# GITS Progress Component

> Reusable Progress Component

## Installation

npm

```
npm i @morpheme/progress
```

yarn

```
yarn add @morpheme/progress
```

pnpm

```
pnpm i @morpheme/progress
```

## Usage

```vue
<script setup lang="ts">
import Progress from '@morpheme/progress';
</script>

<template>
  <Progress />
</template>
```
