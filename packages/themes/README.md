# Morpheme Themes

Morpheme Themes

## Installation

npm

```
npm i @morpheme/themes
```

yarn

```
yarn add @morpheme/themes
```

pnpm

```
pnpm i @morpheme/themes
```

## Usage

```vue
<script setup lang="ts">
import {VThemeProvider} from '@morpheme/themes';
</script>

<template>
  <VThemeProvider theme="morpheme">
    <VAppShell>
      App
    </VAppShell>
  </VThemeProvider>
</template>
```
