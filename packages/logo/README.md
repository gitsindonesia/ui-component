# GITS Logo Component

## Installation

npm

```
npm i @morpheme/logo
```

yarn

```
yarn add @morpheme/logo
```

pnpm

```
pnpm i @morpheme/logo
```

## Usage

```vue
<script setup lang="ts">
import VLogo from '@morpheme/logo';
import '@morpheme/logo/dist/style.css';
</script>

<template>
  <VLogo />
</template>
```
