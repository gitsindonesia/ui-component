# Morpheme Timeline

Morpheme Timeline Component.

## Installation

npm

```
npm i @morpheme/timeline
```

yarn

```
yarn add @morpheme/timeline
```

pnpm

```
pnpm i @morpheme/timeline
```

## Usage

```vue
<script setup lang="ts">
import VTimeline from '@morpheme/timeline';
</script>

<template>
  <VTimeline>
    <VTimelineItem>
      Item 1
    </VTimelineItem>
    <VTimelineItem>
      Item 2
    </VTimelineItem>
    <VTimelineItem>
      Item 3
    </VTimelineItem>
  </VTimeline>
</template>
```
