# GITS UI

Free and open-source UI Component for Vue 3.

## Installation

npm

```
npm i @morpheme/ui
```

yarn

```
yarn add @morpheme/ui
```

pnpm

```
pnpm i @morpheme/ui
```

## Usage

```ts
// file: main.ts

import GitsUI from '@morpheme/ui';
import '@morpheme/ui/styles';
// or load SCSS styles
// import '@morpheme/ui/styles.scss';

const app = createApp(App);

app.use(GitsUI);
```
