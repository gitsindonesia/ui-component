# Morpheme Layouts

> Deprecated

Morpheme Layouts Component.

## Installation

npm

```
npm i @morpheme/layouts
```

yarn

```
yarn add @morpheme/layouts
```

pnpm

```
pnpm i @morpheme/layouts
```

## Usage

```vue
<script setup lang="ts">
import {Auth, Base, Default, SidebarMenu} from '@morpheme/layouts';
</script>

<template>
  <Auth />
  <Base />
  <Default />
  <SidebarMenu />
</template>
```
