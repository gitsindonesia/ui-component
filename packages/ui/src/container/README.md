# Morpheme Container

Morpheme Container Component.

## Installation

npm

```
npm i @morpheme/container
```

yarn

```
yarn add @morpheme/container
```

pnpm

```
pnpm i @morpheme/container
```

## Usage

```vue
<script setup lang="ts">
import Container from '@morpheme/container';
</script>

<template>
  <Container>Text</Container>
</template>
```
