# Morpheme Button Component

> Button Component

## Installation

via npm

```
npm i @morpheme/button
```

via yarn

```
yarn add @morpheme/button
```

via pnpm

```
pnpm i @morpheme/button
```

## Usage

```html
<script setup lang="ts">
  import Button from '@morpheme/button';
</script>

<button color="primary">Click Me</button>
```
