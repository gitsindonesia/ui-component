# Morpheme Spinner Component

> Reusable Spinner Component

## Installation

npm

```
npm i @morpheme/spinner
```

yarn

```
yarn add @morpheme/spinner
```

pnpm

```
pnpm i @morpheme/spinner
```

## Usage

```vue
<script setup lang="ts">
import Spinner from '@morpheme/spinner';
</script>

<template>
  <Spinner />
</template>
```
