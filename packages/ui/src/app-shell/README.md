# Morpheme AppShell

Morpheme Application Shell Component.

## Installation

npm

```
npm i @morpheme/app-shell
```

yarn

```
yarn add @morpheme/app-shell
```

pnpm

```
pnpm i @morpheme/app-shell
```

## Usage

```vue
<script setup lang="ts">
import VAppShell from '@morpheme/app-shell';
</script>

<template>
  <VAppShell>
    Content
  </VAppShell>
</template>
```
