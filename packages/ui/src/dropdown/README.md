# Morpheme Dropdown

Morpheme Dropdown Component.

## Installation

npm

```
npm i @morpheme/dropdown
```

yarn

```
yarn add @morpheme/dropdown
```

pnpm

```
pnpm i @morpheme/dropdown
```

## Usage

```vue
<script setup lang="ts">
import Dropdown from '@morpheme/dropdown';
</script>

<template>
  <Dropdown />
</template>
```
