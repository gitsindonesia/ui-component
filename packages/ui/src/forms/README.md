# Morpheme Forms

Morpheme Forms Component. 

## Installation

npm

```
npm i @morpheme/forms
```

yarn

```
yarn add @morpheme/forms
```

pnpm

```
pnpm i @morpheme/forms
```

## Usage

```vue
<script setup lang="ts">
import {Textarea} from '@morpheme/forms';
</script>

<template>
  <Textarea />
</template>
```
