# Morpheme Command

Morpheme Command Palette Component.

## Installation

npm

```
npm i @morpheme/command
```

yarn

```
yarn add @morpheme/command
```

pnpm

```
pnpm i @morpheme/command
```

## Usage

```vue
<script setup lang="ts">
import {VCommand} from '@morpheme/command';
</script>

<template>
  <VCommand />
</template>
```
