# GITS Menu Component

> Reusable Menu Component

## Installation

npm

```
npm i @morpheme/menu
```

yarn

```
yarn add @morpheme/menu
```

pnpm

```
pnpm i @morpheme/menu
```

## Usage

```vue
<script setup lang="ts">
// import styles
import '@morpheme/menu/dist/style.css';
// import component
import VMenu from '@morpheme/menu';
</script>

<template>
  <VMenu />
</template>
```
