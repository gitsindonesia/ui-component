# GITS Services Component

> Reusable Services Component

## Installation

npm

```
npm i @morpheme/services
```

yarn

```
yarn add @morpheme/services
```

pnpm

```
pnpm i @morpheme/services
```

## Usage

```vue
<script setup lang="ts">
import Services from '@morpheme/services';
</script>

<template>
  <Services />
</template>
```
