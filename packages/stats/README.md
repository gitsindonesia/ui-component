# GITS Stats Component

> Reusable Stats Component

## Installation

npm

```
npm i @morpheme/stats
```

yarn

```
yarn add @morpheme/stats
```

pnpm

```
pnpm i @morpheme/stats
```

## Usage

```vue
<script setup lang="ts">
import Stats from '@morpheme/stats';
</script>

<template>
  <Stats />
</template>
```
