# GITS Menus Component

> Reusable Menus Component

## Installation

npm

```
npm i @morpheme/menus
```

yarn

```
yarn add @morpheme/menus
```

pnpm

```
pnpm i @morpheme/menus
```

## Usage

```vue
<script setup lang="ts">
// import styles
import '@morpheme/menus/dist/style.css';
// import component
import Menus from '@morpheme/menus';
</script>

<template>
  <Menus />
</template>
```
