# GITS Icon Component

## Installation

npm

```
npm i @morpheme/icon
```

yarn

```
yarn add @morpheme/icon
```

pnpm

```
pnpm i @morpheme/icon
```

## Usage

```vue
<script setup lang="ts">
import {Icon} from '@morpheme/icon';
</script>

<template>
  <Icon icon="ic:add" />
</template>
```
