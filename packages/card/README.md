# GITS Card Component

> Reusable Card Component

## Installation

via npm

```
npm i @morpheme/card
```

via yarn

```
yarn add @morpheme/card
```

via pnpm

```
pnpm i @morpheme/card
```

## Usage

```vue
<script setup lang="ts">
import Card from '@morpheme/card';
</script>

<template>
  <Card title="Header">My Body</Card>
</template>
```
