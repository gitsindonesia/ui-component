# GITS Badge Component

> Badge Component

## Installation

npm

```
npm i @morpheme/badge
```

yarn

```
yarn add @morpheme/badge
```

pnpm

```
pnpm i @morpheme/badge
```

## Usage

```vue
<script setup lang="ts">
import Badge from '@morpheme/badge';
</script>

<template>
  <Badge color="primary"> Text </Badge>
</template>
```
