# GITS Spinner Component

> Reusable Spinner Component

## Installation

npm

```
npm i @gits-id/spinner
```

yarn

```
yarn add @gits-id/spinner
```

pnpm

```
pnpm i @gits-id/spinner
```

## Usage

```vue
<script setup lang="ts">
import Spinner from '@gits-id/spinner';
</script>

<template>
  <Spinner />
</template>
```
