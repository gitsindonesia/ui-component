# GITS Navbar Component

> Reusable Navbar Component

## Installation

npm

```
npm i @morpheme/navbar
```

yarn

```
yarn add @morpheme/navbar
```

pnpm

```
pnpm i @morpheme/navbar
```

## Usage

```vue
<script setup lang="ts">
import Navbar from '@morpheme/navbar';
</script>

<template>
  <Navbar />
</template>
```
