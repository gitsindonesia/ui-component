## Styling

Morpheme UI uses SASS and CSS Variables as main styles.

## Browser Support

Morpheme UI supports all major browsers except IE 11 and Opera Mini.


## Vue and Nuxt

Morpheme UI **only** support Vue 3 and Nuxt 3. No plan to support Vue 2 for now.
